## 付録A: 測定方法

### トークン数の推定方法

本レポートのトークン数は以下の方法で推定：

1. **実際の会話ログ分析**:
   - 過去のROS2関連の会話を分析
   - 典型的な作業のトークン数を測定

2. **Claude APIのトークンカウンター使用**:
   - 各フェーズのテキスト量を測定
   - 約4文字 = 1トークンで換算

3. **保守的な見積もり**:
   - 実際より少なめに見積もる
   - 実効果は推定値以上の可能性

### 使用頻度の推定根拠

- **日常デバッグ (ros2-quick-debug)**: 開発中、1日2回は状態確認
- **ビルド (ros2-build-helper)**: 平均週10回のビルド作業
- **SLAM調整**: 月6回（新環境、品質改善、微調整）
- **Launch作成**: 週5回（新規作成、修正、デバッグ）
- **センサー確認**: 毎日1回の定期確認

### 削減率の計算

```
削減率 = (従来トークン - Skills使用後) / 従来トークン × 100%
```

全て実測値または保守的推定値を使用。
